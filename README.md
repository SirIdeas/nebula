# Ejemplo Nébula - Control completo de plataforma NebualBot.

Aplicación ejemplo para el control de todos los componentes integrados en la plataforma NebulaBot.

## Requerimientos

Descargar [Nébula Librería Android](https://github.com/SirIdeas/nebula/archive/android-stable-1.0.0.zip).

## Instalación

- Descargar código fuente de la aplicación.
- Agrgar librería Nebula descargada.
